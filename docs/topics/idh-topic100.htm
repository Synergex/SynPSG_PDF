AutoImage Class

 

This class defines an image that is to be drawn on one or more pages of a PdfFile object using absolute coordinates.

 

Inheritance Heirarchy

 

System.Object

  SynPSG.AutoItem

    SynPSG.PDF.AutoImage

 

Namespace: SynPSG.PDF

 

Syntax

 

public class AutoImage extends AutoItem

 

The AutoImage class exposes the following members:

 

Constructors

 

The AutoImage class has a single constructor method that has several required and optional parameters:

 

public method AutoImage

          required in aFileSpec, string

          required in aX, float

          required in aY, float

          optional in aWidth, float

          optional in aHeight, float

          optional in aMode, AutoItemMode

          endparams

 

For guidance on constructing objects of this type refer to PdfFile.DrawImage.

 

Properties

 

FileSpec

string

REQUIRED. Defines the file specification of the image to display. JPEG (.jpg or .jpeg) and PNG (.png) files are supported. Specify a full or relative path to a supported image file. Logical names are supported.

Height

float

Defines the height to be used to draw the image in device independent pixels. Defaults to the pixel height of the image.

Mode

AutoItemMode

Defines whether the box should be drawn at the beginning or end of the processing of each page.

Width

float

Defines the width to be used to draw the image in device independent pixels. Defaults to the pixel width of the image.

X

float

REQUIRED. Defines the X coordinate where the bottom right pixel of the image should be positioned.

Y

float

REQUIRED. Defines the Y coordinate where the bottom right pixel of the image should be positioned.

 

Methods

 

None

 

Remarks

 

Instances of this class can be added to the PdfFile.AutoItems collection or can be passed to the PdfFile.DrawImageObject method.
