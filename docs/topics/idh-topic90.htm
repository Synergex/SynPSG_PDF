AutoTextRowCol Class

 

This class defines text that is to be drawn on one or more pages of a PdfFile object using row/column coordinates.

Properties

 

Bold

boolean

Defines whether the text should be painted using a bold font.

Col

int

Defines the column coordinate of the cell where the first character of the text should be drawn.

Color

PdfColor

Defines a custom color to use when drawing the text.

Italic

boolean

Defines whether the text should be painted using an italic font.

Mode

AutoItemMode

Defines whether the text should be drawn at the beginning or end of the processing of each page.

NewLine

boolean

Defines whether the current position should be set to the first column on the next row after drawing the text.

Row

int

Defines the row coordinate of the cell where the first character of the text should be drawn.

Text

string

REQUIRED. Defines the text to be drawn.

Underline

boolean

Defines whether the text should be underlined.

 

Methods

 

None

 

Remarks

 

Instances of this class can be added to the PdfFile.AutoItems collection or can be passed to the PdfFile.DrawTextObjectRC method.
