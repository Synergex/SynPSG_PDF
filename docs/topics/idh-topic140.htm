Working with Lines

 

There are two ways to work with lines, either via the row/column coordinates system or via the absolute coordinates system. These two mechanisms are not mutually exclusive; you can work with both if you need to, even within a single page.

 

Drawing Lines via Row/Column Coordinates

 

The row/column coordinates system is based on the minimum number of columns that you defined when you called the constructor method. If you didn't specify a minimum number of columns then the default value of 80 would have been used. Based on this minimum number of columns, and other factors such as margins and page orientation.

 

The PdfFile class has specific methods that allow you to work with lines and the row/column coordinates system. These methods are:

 

•DrawLineRC

•DrawLineObjectRC

 

Here is some example code that displays lines using row/column coordinates:

 

;;Create an instance of the PdfFile class

pdf = new PdfFile()

 

;;Draw a thick blue horizontal line across the current row

pdf.DrawLineRow(,10,PdfColor.Blue)

 

;;Draw a vertical line all the way down the left column

pdf.DrawLineRC(1,1,pdf.Rows,1)

 

;;Draw a horizontal line across the center of the page

pdf.DrawLineRC((pdf.Rows/2),1,(pdf.Rows/2),pdf.Columns)

 

;;Draw a diagonal line from bottom/left to top/right

pdf.DrawLineRC(pdf.Rows,1,1,pdf.Columns)

 

;;Show the results

pdf.Preview()

 

Drawing Lines via Absolute Coordinates

 

The absolute coordinates system is based on the physical dimensions of the page expressed in device independent pixels. Any margins or page headers and/or footers that you may have configured do not apply when working with absolute coordinates.

 

The PdfFile class has specific methods that allow you to work with lines and the absolute coordinates system. These methods are:

 

•DrawLine

•DrawLineObject

 

Here is some example code that displays lines using absolute coordinates:

 

;;Create an instance of the PdfFile class

pdf = new PdfFile()

 

;;Draw a thick blue horizontal line across the top of the page

pdf.DrawLine(0,pdf.Height-10,pdf.Width,pdf.Height-10,10,PdfColor.Blue)

 

;;Draw a vertical line all the way down the center of the page

pdf.DrawLine((pdf.Width/2),pdf.Height,(pdf.Width/2),0)

 

;;Draw a diagonal line from bottom/left to top/right

pdf.DrawLine(0,0,pdf.Width,pdf.Height)

 

;;Show the results

pdf.Preview()

 

Auto Line Items

 

Auto line items are line objects that you can define once and that will then be added automatically to each page. For additional information refer to working with auto items.
