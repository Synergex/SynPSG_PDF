Working with Auto Items

 

Auto items are instances of text, lines, boxes or images that are automatically added to each page of a PDF document. There are two types of auto item, these are called "begin page" auto items and "end page" auto items. Begin page auto items are added to each page as soon as the page is created; they are the very first things added to a page. End page auto items are added to a page at the end of the processing of that page; they are the very last things added to a page. This can be important because it is possible that begin page auto items could have things drawn over the top of them, whereas this can never happen with end page auto items (except for other end page auto items). By default auto items are end page auto items, unless you determine otherwise.

 

The PdfFile.AutoItems Property

 

Auto items are defined by adding various types of auto item objects to the PdfFile.AutoItems property which exposes a "collection". There are several types of auto items:

 

AutoTextRowColumn

This class defines an item of text that is to be printed on a PDF page using row/column coordinates.

AutoTextAbsolute

This class defines an item of text that is to be printed on a PDF page using absolute coordinates.

AutoLineRowColumn

This class defines a line that is to be drawn on a PDF page using row/column coordinates.

AutoLineAbsolute

This class defines a line that is to be drawn on a PDF page using absolute coordinates.

AutoBoxRowColumn

This class defines a box that is to be drawn on a PDF page using row/column coordinates.

AutoBoxAbsolute

This class defines a box that is to be drawn on a PDF page using absolute coordinates.

AutoImage

This class defines an image that is to be drawn on a PDF page using absolute coordinates.

 

Within their respective begin page or end page groupings, auto items are always processed in the order in which they are added to the auto items collection.

 

Auto Text Items

 

Auto text are text items that are automatically printed on each page. There are two types of auto text items:

 

•AutoTextRowColumn

•AutoTextAbsolute

 

 

Auto Line Items

 

Auto lines are lines that are automatically drawn on each page. There are two types of auto line items:

 

•AutoLineRowCol

•AutoLineAbsolute

 

Auto Box Items

 

Auto boxes are boxes that are automatically drawn on each page. There are two types of auto box items:

 

•AutoBoxRowCol

•AutoBoxAbsolute

 

Auto Image Items

 

Auto images are images that are automatically drawn on each page. There is only one auto image item:

 

•AutoImage

 

Auto Item Mode

 

Each auto item object has a property called Mode that accepts a value defined by the AutoItemMode enumeration. This value defines whether the auto item is added to the page at the beginning or end of the processing of the page.

 

Auto items that are added at the beginning of the page could potentially be overlaid by other items drawn on the page at a later time. Auto items added during end of page processing are the last items that will be drawn on a page, and therefor could overlay other items drawn earlier.

 

The decision of whether to use BeginPage or EndPage auto items depends on your specific requirements. Auto items default to being EndPage auto items.
