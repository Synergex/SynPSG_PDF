PdfPageSize Enumeration

 

This enumeration defines the options for the size of pages in a PdfFile object.

 

    public enum PdfPageSize

        LETTER,         HPdfPageSizes.HPDF_PAGE_SIZE_LETTER

        LEGAL,          HPdfPageSizes.HPDF_PAGE_SIZE_LEGAL

        A3,             HPdfPageSizes.HPDF_PAGE_SIZE_A3

        A4,             HPdfPageSizes.HPDF_PAGE_SIZE_A4

        A5,             HPdfPageSizes.HPDF_PAGE_SIZE_A5

        B4,             HPdfPageSizes.HPDF_PAGE_SIZE_B4

        B5,             HPdfPageSizes.HPDF_PAGE_SIZE_B5

        EXECUTIVE,      HPdfPageSizes.HPDF_PAGE_SIZE_EXECUTIVE

        US4x6,          HPdfPageSizes.HPDF_PAGE_SIZE_US4x6

        US4x8,          HPdfPageSizes.HPDF_PAGE_SIZE_US4x8

        US5x7,          HPdfPageSizes.HPDF_PAGE_SIZE_US5x7

        COMM10,         HPdfPageSizes.HPDF_PAGE_SIZE_COMM10

        EOF,            HPdfPageSizes.HPDF_PAGE_SIZE_EOF

    endenum
