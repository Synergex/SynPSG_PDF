Working with Boxes

 

There are two ways to work with boxes, either via the row/column coordinates system or via the absolute coordinates system. These two mechanisms are not mutually exclusive; you can work with both if you need to, even within a single page.

 

Drawing Boxes via Row/Column Coordinates

 

The row/column coordinates system is based on the minimum number of columns that you defined when you called the constructor method. If you didn't specify a minimum number of columns then the default value of 80 would have been used. Based on this minimum number of columns, and other factors such as margins and page orientation.

 

The PdfFile class has specific methods that allow you to work with boxes and the row/column coordinates system. These methods are:

 

•DrawBoxRC

•DrawBoxObjectRC

 

Here is some example code that displays boxes using row/column coordinates:

 

;;Create an instance of the PdfFile class

pdf = new PdfFile()

 

;;Draw a box from the current position (1,1) to the bottom/right corner.

pdf.DrawBoxRC()

 

;;Draw an "Atomic Tangerine" colored box in the upper left of the page

pdf.DrawBoxRC(5,5,10,30,,PdfColor.AtomicTangerine)

 

;;Draw a filed "Antique Brass" colored box in the upper right of the page

pdf.DrawBoxRC(5,45,10,70,,PdfColor.AntiqueBrass,true,PdfColor.AntiqueBrass)

 

;;Show the results

pdf.Preview()

 

Drawing Boxes via Absolute Coordinates

 

The absolute coordinates system is based on the physical dimensions of the page expressed in device independent pixels. Any margins or page headers and/or footers that you may have configured do not apply when working with absolute coordinates.

 

The PdfFile class has specific methods that allow you to work with boxes and the absolute coordinates system. These methods are:

 

•DrawBox

•DrawBoxObject

 

Here is some example code that displays boxes using absolute coordinates:

 

;;Create an instance of the PdfFile class

pdf = new PdfFile()

 

;;Draw a box around the whole page, 5 pixels in from the edge

pdf.DrawBox(5,5,pdf.Width-5,pdf.Height-5)

 

;;Draw an "Atomic Tangerine" colored box in the upper left of the page

pdf.DrawBox(50,750,300,550,,PdfColor.AtomicTangerine)

 

;;Draw a filed "Antique Brass" colored box in the upper right of the page

pdf.DrawBox(325,750,575,550,,PdfColor.AntiqueBrass,true,PdfColor.AntiqueBrass)

 

;;Show the results

pdf.Preview()

 

Auto Box Items

 

Auto box items are box objects that you can define once and that will then be added automatically to each page. For additional information refer to working with auto items.
