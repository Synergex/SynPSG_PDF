AutoBoxAbsolute Class

 

This class defines a box that is to be drawn on one or more pages of a PdfFile object using absolute coordinates.

Properties

 

FromX

float

REQUIRED. Defines the X coordinate of one corner of the box to be drawn.

FromY

float

REQUIRED. Defines the Y coordinate of one corner of the box to be drawn.

Fill

boolean

Defines whether the box should be filled with the fill color. Default is not to fill the box.

FillColor

PdfColor

Defines a custom fill color to be used if filling the box.

Mode

AutoItemMode

Defines whether the box should be drawn at the beginning or end of the processing of each page.

StrokeColor

PdfColor

Defines a custom stroke color to use when drawing the box.

ToX

float

REQUIRED. Defines the X coordinate of the opposite corner of the box to be drawn.

ToY

float

REQUIRED. Defines the Y coordinate of the opposite corner of the box to be drawn.

Width

float

Defines the width of the line used to draw the outline of the box.

 

Methods

 

None

 

Remarks

 

Instances of this class can be added to the PdfFile.AutoItems collection or can be passed to the PdfFile.DrawBoxObject method.

 

Examples

 

This source code adds an AutoBoxAbsolute to the AutoItems collection of a PDF document. The box will be drawn on each page.

 

;;Create an instance of the PdfFile class

pdf = new PdfFile()

 

;;Define an auto box to be drawn on each page

pdf.AutoItems.Add(new AutoBoxAbsolute(100,550,300,450,3,PdfColor.Orange,true))

 

;;Add three pages to the document, each should have the box

pdf.NewPage()

pdf.NewPage(PdfOrientation.Landscape)

pdf.NewPage()

 

;;Show the results

pdf.Preview()
