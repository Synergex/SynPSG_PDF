AutoBoxRowCol Class

 

This class defines a box that is to be drawn on one or more pages of a PdfFile object using row/column coordinates.

Properties

 

FromRow

int

Defines the row of one corner of the box. Defaults to the current row.

FromCol

int

Defines the column of one corner of the box. Defaults to the current column.

Fill

boolean

Defines whether the box should be filled with the fill color. Default is not to fill the box.

FillColor

PdfColor

Defines a custom fill color to be used if filling the box.

Mode

AutoItemMode

Defines whether the box should be drawn at the beginning or end of the processing of each page.

StrokeColor

PdfColor

Defines a custom stroke color to use when drawing the box.

ToRow

int

Defines the row of the opposite corner of the box. Defaults to the highest row.

ToCol

int

Defines the column of the opposite corner of the box. Defaults to the highest column.

Width

float

Defines the width of the line used to draw the outline of the box.

 

Methods

 

None

 

Remarks

 

Instances of this class can be added to the PdfFile.AutoItems collection or can be passed to the PdfFile.DrawBoxObjectRC method.
