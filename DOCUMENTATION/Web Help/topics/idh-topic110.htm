AutoTextAbsolute Class

 

This class defines text that is to be drawn on one or more pages of a PdfFile object using absolute coordinates.

For guidance on constructing objects of this type refer to PdfFile.DrawText.

 

Properties

 

Color

PdfColor

Defines a custom color to use when drawing the text.

FontName

string

Defines a custom font name to be used to draw the text. The font name must be the name of one of the implicit fonts, or the internal name of a TrueType font loaded by LoadTrueTypeFont.

FontSize

int

Defines a custom font size to be used to draw the text.

Mode

AutoItemMode

Defines whether the text should be drawn at the beginning or end of the processing of each page.

Text

string

REQUIRED. Defines the text to be drawn.

boolean

Defines whether the text should be underlined.

X

float

REQUIRED. Defines the X coordinate at which the the bottom right corner of the text should be positioned.

Y

float

REQUIRED. Defines the Y coordinate at which the the bottom right corner of the text should be positioned.

 

Methods

 

None

 

Remarks

 

Instances of this class can be added to the PdfFile.AutoItems collection or can be passed to the PdfFile.DrawTextObject method.
