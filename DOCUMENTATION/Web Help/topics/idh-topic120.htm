Creating a PDF File

 

To create a new PDF document you create an instance of the PdfFile class and then interact with the resulting object in order to define the content of the document.

 

Creating an Instance of the PdfFile Class

 

Creating an instance of the PdfFile class establishes the basic context that can be used to create a PDF document. For example:

 

import SynPSG.PDF

 

main PdfTest

    record

        pdf, @PdfFile

    endrecord

proc

    pdf = new PdfFile()

   

   

    pdf = ^null

endmain

 

In the above example the PDF document created would be configured to contain letter sized pages and the default page orientation would be set to portrait. When addressing paged in the document using the row/column coordinates system each page would have an initial margin of 0.16" all the way around. No page header or footer lines would be defined, and the page body would be organized into an 80-column wide grid. If necessary the left and right margins would then be adjusted in order to ensure that the 80-column wide page body was centered horizontally.

 

The PdfFile class constructor method defines several optional parameters that you can manipulate in order to configure various options. Below you will find several examples of doing so:

 

Page Size

 

The following code creates a PDF document that will contain A4 pages.

 

pdf = new PdfFile(PdfPageSize.A4)

 

Page Orientation

 

The following code creates a PDF document in which pages will default to landscape orientation, but portrait pages could still be added.

 

pdf = new PdfFile(,PdfOrientation.Landscape)

 

Available Columns

 

The following code creates a PDF document that will have 100 columns available in the page body. The page size will be letter and pages will default to portrait orientation, but landscape paged could still be added.

 

pdf = new PdfFile(,,100)

 

Page Margins

 

The following code creates a PDF document whose pages will have 0.75" left and right margins and 1.5" top and bottom margins. Pages will be letter size and will default to portrait orientation. The page body will be 60 columns wide.

 

pdf = new PdfFile(,,60,,,0.75,1.5,0.75,1.5)

 

Header and Footer Rows

 

The following code creates a PDF document whose letter pages will default to landscape orientation and the page body will be organized into 132 columns. There will be two page header rows and two page footer rows.

 

pdf = new PdfFile(,PdfOrientation.Landscape,132,,,,,,,2,2)

 

There are lots of other possible combinations. For further information refer to the PdfFile class constructor method.

 

Setting Preferred Page Layout

 

PDF reader applications can present the pages of PDF documents in a number of different ways, e.g. display a whole page at a time, display the whole width of a single page, display two pages side-by-side, etc.

 

You can set a preference for how PDF viewer applications display your document by setting PdfFile.PageLayout using a value from the PdfPageLayout enumeration . E.g.:

 

pdf = new PdfFile()

pdf.PageLayout = PdfPageLayout.Single

 

 

Setting Preferred Display Mode

 

PDF reader applications typically support several different views that can be used to present the pages of PDF documents, e.g. display the document with a thumbnails window, display the document with an outline view, etc.

 

You can set a preference for the view that a PDF viewer application should use by setting PdfFile.PageMode using a value from the PdfPageLayout enumeration . E.g.:

 

pdf = new PdfFile()

pdf.PageMode = PdfPageMode.Outline
